7. 燃費計算ロジック
uel Efficiency=
Fuel
Distance
	

計算例
Distance = 40 km
Fuel = 2 L
Result = 20 km/L

計算ロジック
if (fuelLiter > 0)
{
    fuelEfficiency = distanceKm / fuelLiter;
}
else
{
    throw new Exception("Fuel must be greater than 0.");
}
