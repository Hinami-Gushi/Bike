## 1. 背景・目的

### 1.1 背景

バイク利用者は給油のたびに走行距離・給油量・燃料費をメモ帳やスプレッドシートで管理していることが多く、燃費の推移や月ごとのコストを把握しにくい状況にある。スマートフォンからすぐに記録・確認できる軽量な Web アプリが求められている。

### 1.2 目的

- 給油情報をその場でスマートフォンから記録できるようにする
- 燃費（km/L）を自動計算し、確認の手間をなくす
- 月ごとの燃料費を可視化し、コスト管理に役立てる
- 将来的な複数車両対応・グラフ機能の拡張に耐えられる構成にする

## 3. 機能要件

### 3.1 機能一覧（MVP）

| ID | 機能名 | 概要 | 認証 |
|----|--------|------|------|
| F-01 | 給油記録追加 | 給油日・給油量・走行距離・燃料費を登録 | 不要（v0.1） |

## 4. 入力・出力仕様

### 4.1 給油記録の入力項目

| 項目名 | 型 | 必須 | 説明 |
|--------|----|------|------|
| Date | date | ○ | 給油日 |
| Fuel Amount | number | ○ | 給油量（L）。0 より大きい値 |
| Distance | number | ○ | 走行距離（km）。0 より大きい値 |
| Cost | number | △ | 燃料費（円）。省略可 |

### 4.2 燃費計算ロジック

```
Fuel Efficiency (km/L) = Distance / Fuel Amount
```

- `Fuel Amount` が 0 以下の場合はエラーを表示し保存しない
- 表示は小数点第 1 位まで（例：`20.3 km/L`）

### 4.3 目的地別給油試算ロジック（F-08）

| 計算項目 | 計算式 |
|----------|--------|
| 必要燃料量（L） | `目的地までの距離 ÷ 参照燃費` |
| 推定燃料費（円） | `必要燃料量 × 1L あたりの燃料単価` |

**参照燃費の優先順位：**

1. ユーザーが手動入力した燃費値（任意）
2. 直近 3 件の給油記録の平均燃費（自動補完）
3. 記録が存在しない場合はエラー表示し試算不可

**燃料単価の入力について：**

- ユーザーが試算画面で手動入力する（例：¥175/L）
- 最後に給油記録へ入力した単価を初期値としてプリフィル表示する
  - 単価 = `cost ÷ fuel_liter`（端数切り捨て）
  - 燃料費（cost）が未入力の給油記録は単価の計算対象外

**バリデーション：**

- 距離・燃料単価は 0 より大きい数値
- 参照燃費が存在しない場合は「燃費データがありません。先に給油記録を追加してください」と表示

### 5.1 ダッシュボード（`/`）

```
[FuelLog]
├─ Average Fuel Efficiency:  20.3 km/L
├─ Monthly Fuel Cost (May):  ¥4,500
├─ [+ Add Fuel Log]      →  /add へ遷移
└─ [Trip Calculator]     →  /trip へ遷移
```

### 5.2 給油記録追加画面（`/add`）

```
[Add Fuel Log]
├─ Date          [____-__-__]
├─ Fuel Amount   [____] L
├─ Distance      [____] km
├─ Cost          [____] ¥  (optional)
└─ [Save]  /  [Cancel]
```

バリデーション：
- Date・Fuel Amount・Distance は必須
- Fuel Amount・Distance は 0 より大きい数値
- エラーはフォーム上部にインライン表示

### 5.3 給油履歴一覧（`/logs`）

```
[Fuel Log History]
├─ 2026/05/10 | 40 km / 2.0 L | 20.0 km/L | ¥300   [Edit] [Delete]
├─ 2026/05/05 | 35 km / 1.8 L | 19.4 km/L | -       [Edit] [Delete]
└─ ...
```

### 5.4 給油記録編集画面（`/edit/:id`）

追加画面と同じフォーム構成。既存値をプリフィル表示する。

### 5.5 目的地別給油試算画面（`/trip`）

```
[Trip Fuel Calculator]
├─ Distance to Destination   [____] km          ← 必須
├─ Fuel Price per Liter      [____] ¥/L          ← 必須（最終記録から自動補完）
├─ Override Fuel Efficiency  [____] km/L (opt)   ← 空欄なら直近3件の平均を使用
├─ [Calculate]
│
└─ [Result]（Calculate 押下後に表示）
    ├─ Fuel Needed:   2.1 L
    ├─ Est. Cost:     ¥368
    └─ Based on:      20.3 km/L (avg of last 3 logs)
```

- 試算結果はページ遷移なしで同一画面に表示する（JavaScript で動的表示）
- 試算結果は保存しない（画面上の表示のみ）
- ダッシュボードに「Trip Calculator」ボタンを追加し `/trip` へ遷移





### 6.1 `fuel_logs` テーブル

| カラム名 | 型 | 説明 |
|----------|----|------|
| id | UUID (PK) | 一意 ID |
| date | date | 給油日 |
| fuel_liter | decimal(6,2) | 給油量（L） |
| distance_km | decimal(8,2) | 走行距離（km） |
| fuel_efficiency | decimal(6,2) | 燃費（km/L）。保存時に自動計算 |
| cost | integer | 燃料費（円）。NULL 許容 |
| created_at | timestamptz | 作成日時 |
| updated_at | timestamptz | 更新日時 |

## 7. 非機能要件

| 項目 | 内容 |
|------|------|
| 対応端末 | Android / iOS のモバイルブラウザ中心。PC ブラウザも動作すること |
| UI 言語 | English（v0.1 固定） |
| レスポンシブ | スマートフォン幅（375px〜）に最適化 |
| レスポンス | 一覧・ダッシュボード表示：ローカル環境で 2 秒以内 |
| データ保存 | クラウド DB（PostgreSQL / Firestore のいずれか） |
| アプリ形式 | Web アプリ（PWA 対応は v0.2 以降） |
| パスワード | v0.1 は認証なし |
